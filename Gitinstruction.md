# Работа с Git

## 1. Проверка наличия установленного Git

   В терминале выполнить команду `git --version`
Если Git установлен появится сообщение о версии программы, иначе будет сообщение об ошибке 

## 2. Установка Git

Загружаем последнюю версию Git с сайта " https://git-scm.com/downloads " Устанавливаем с настройками по умолчанию.

## 3. Настройка Git

При первом использовании Git необходимо представиться. Для этого нужно ввести в терминале две команды :
```
 git config --global user.name «Ваше имя английскими буквами» 

git config --global user.email ваша почта@example.com 
```
## 4. Инициализация репозитория

Для того, чтобы Git смог работать с нашей папкой, необходимо открыть папку в которй это будет происходить и ввести команду в теминале ( `git init`)
Git создает в ней репозиторий.

## 5. Запись изменений в репозитории

Чтобы записать внесенные изменения необходимо ввести команду (`git add`) чтобы добавить файлы к последующему коммиту. затем создать коммит с комментарием, чтобы в дальнейшем было удобнее понимать, какие изменения внесены. Для этого набираем в терминале команду (`git commit`)
